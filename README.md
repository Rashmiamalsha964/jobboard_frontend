# 💼 JobBoard Frontend

This is the frontend of the JobBoard application built using **Next.js (React)** and **Tailwind CSS**.

---

## 🚀 Features

- View all job posts
- Create new job requests
- Category filtering
- Responsive UI (mobile + desktop)
- API integration with backend

---

## 🛠️ Tech Stack

- Next.js
- React.js
- Tailwind CSS
- Axios / Fetch API

---
## ⚙️ Setup Instructions

### 1. Clone Repository
```bash
git clone https://github.com/Rashmiamalsha964/jobboard_frontend.git

cd jobboard-frontend

2. Installation

```bash
npm install

▶️ Run Locally
npm run dev

App runs on:

http://localhost:3000


🔗 Environment Variables
Create .env.local file:
NEXT_PUBLIC_API_URL=http://localhost:5000/api

For production (Vercel):
NEXT_PUBLIC_API_URL=https://your-backend-url.com/api


🌐 Deployment
Deploy on Vercel
Set environment variables in Vercel dashboard

📡 API Connection

Frontend connects to backend for:

Create Job
Get Jobs
Job Details
